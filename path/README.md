# Appointment Scheduling System

A production-ready, full-stack Appointment Scheduling System.

## Technology Stack

- **Frontend:** React (Create React App)
- **Backend:** Node.js (Express)
- **Database:** MongoDB
- **Authentication:** JWT

## Structure

- `/backend` — Node.js/Express API server
- `/frontend` — React SPA
